# Azure VNet Design & Implementation

### [Link to Demonstrations and Labs](https://emlykf.gitbook.io/azure-vnet-design-and-implementation/)

## Project Introduction

This project provides an overview of Azure networking fundamentals, showcasing my understanding and practical application.

## What We Will Learn

* Understanding Azure VNet
* Planning VNet naming and location
* Planning VNet IP address spaces
* Planning VNet subnet segmentation
* Hands-on lab: creating a single-stack VNet in Azure
* Hands-on lab: creating a dual-stack VNet in Azure
* Understanding private IP address assignment for subnet workloads
* Hands-on lab: determining the VM location and sizes for future labs
* Hands-on lab: exploring private IP assignments

## Prerequisites

* A PC with an internet connection
* An active Azure subscription
